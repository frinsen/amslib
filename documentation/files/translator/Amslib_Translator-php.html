translator

Amslib_Translator.php

description

Translate strings from between a recognised input and output, it’s fast, flexible and easy to use (easier than PO files anyway)

The translator object uses a language catalog like gettext but doesnt really suck by actually doing things cleverly, you can learn, forget, translate mix catalogues from different parts of projects together, it’s much more powerful and does exactly the same job.
